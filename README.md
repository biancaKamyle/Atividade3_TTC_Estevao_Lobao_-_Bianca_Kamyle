## Informações Base

O módulo LoRa-02, que utiliza o chip SX1278, opera na banda de frequência ISM entre 410 MHz e 525 MHz. No Brasil, embora a frequência mais usada para tecnologias LoRa seja a de 915 MHz, optamos pela utilização de **433 MHz**, visto que esta é a frequência de operação do chip e o padrão definido pelo fabricante para este módulo.
